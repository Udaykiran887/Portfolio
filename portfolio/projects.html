My Projects

  • Project 1: Daily Weather View on GitHub
    This project demonstrates how to create a Daily Weather Dashboard using Node-RED and WeatherAPI. The dashboard fetches real-time weather data for a user-selected city and displays current temperature, weather condition, wind speed, and humidity. Ensuring users have up-to-date weather information. The dashboard is created using Node-RED Dashboard nodes for UI, and it connects to the WeatherAPI to fetch live weather data..
  • Project 2: Random Quote Generator View on GitHub
    The Random Quote Generator is a Node-RED flow that integrates with the API-Ninjas Quotes API to fetch random quotes based on specific categories. The quotes are displayed on a Node-RED dashboard, and the flow can be configured to save quotes into a MySQL database for later use. This project is perfect for users who want to learn about API integration and build an interactive, customizable quote generation tool.

  • Project 4: Current Time View on GitHub
    The Current Time App is a simple Node.js (or Node-RED) application that uses the TimeZoneDB API to retrieve the current time for any selected time zone. This project allows users to query the time for a specific zone and format the response data, including details such as the country, GMT offset, and daylight saving time (DST) status.
